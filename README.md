##  Tech Stack

| Category | Technologies |
|-----------|---------------|
|  Framework | React 18 + Vite |
|  Styling | TailwindCSS + Ant Design |
|  Language | TypeScript |
|  Networking | Axios with interceptors |
|  State | Zustand |
|  Charts | Recharts |
|  Animations | Framer Motion |
|  Linting | ESLint + TS rules |

---

##  Features

-  **Fake Login** & protected routes  
-  **Crypto & NFT pages** with live or mock data  
-  **News feed** from CryptoPanic API  
-  **Favorites** system (local)  
-  **PnL & Investment simulators**   
-  **Automatic fallback to mock data** when API fails or CORS blocks requests
